# Flash-Foley!

Welcome to the repo for Flash-Foley: Fast Interactive Sketch2Audio Generation! This repo holds all code for running Flash-Foley in a controlled Gradio environment, as well as training and accelerating your own sketch-conditioned audio models. This code borrows heavily from the `stable-audio-tools` codebase, so shoutout and thanks to the great harmonai folks there!


# Install

To run the training scripts or inference code, you'll want to clone this repository, navigate to the root, and run:
```bash
$ pip install .
```

# Requirements
Requires PyTorch 2.5 or later for Flash Attention and Flex Attention support

Development for the repo is done in Python 3.10
